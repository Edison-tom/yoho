# Yoho 宠物互动效果设计

> Yoho macOS 端 P2 宠物模块 | 2026-07-04
> 4 品种 × 12+ 场景 = ~48 组独立动画


## 一、渲染方案

### MVP：Alpha MP4 视频精灵
- 格式：HEVC with Alpha (iOS 13+/macOS 10.15+ 原生支持)
- 单文件 50-200KB，总素材包 < 12MB
- 播放器：`AVPlayer` + `AVPlayerLayer`，循环/单次切换
- 代码层粒子叠加：❤️/⭐/✨ 等 SF Symbol 叠加在视频上层

### 后续可选升级
- Spine 2D 骨骼动画（美术工作流切换成本高）
- 当前 MP4 方案满足 MVP 需求，≥ 1.0 上线后评估

## 四、粒子效果

| 场景 | 粒子类型 | 颜色 | 数量 | 动画 |
|---|---|---|---|---|
| 摸摸头 | ❤️ heart.fill | 粉色/红色 | 3-5 | 从宠物头上冒出上飘消散 |
| 投喂 | ✨ sparkle | 金色 | 4-6 | 饼干碎裂飞散 |
| 排泄 | ⭐ star.fill | 金色 | 1-2 | 从宠物身后弹出落下 |

| 结果期 | 🎆 烟花 | 金色 | 全屏 | 中心爆开 |

## 五、代码架构

```
Sources/Yoho/
├── Views/
│   ├── AlphaMP4Player.swift        ← AVPlayer 封装（循环/单次/alpha 支持）
│   ├── PetAnimationView.swift      ← 视频 + 粒子 + 状态切换
│   ├── PetView.swift               ← 交互层（拖拽/点击/长按）
│   └── ParticleOverlay.swift       ← 粒子效果组件
├── Models/
│   └── PetAnimation.swift          ← 动画配置（文件名/循环/时长）
└── Stores/
    └── PetStore.swift              ← 微动作定时器 + 状态管理
```

## 六、MP4 文件规格

- 分辨率：200×200 pt (@2x = 400×400 px)
- 帧率：30fps
- 编码：HEVC (H.265) with Alpha
- 背景：透明
- 对齐：宠物居中，留 10% 边距

## 七、占位方案

在 MP4 素材到位前，每个品种 × 状态用 SwiftUI 原生动画占位：

- 品种差异：不同渐变色 + 体型比例
- 状态差异：缩放/旋转/位移/不透明度组合
- 粒子效果：SF Symbol + SwiftUI Animation

占位动画与正式 MP4 接口完全一致（`PetAnimationView` 判断 bundle 中是否有对应 MP4，有则播放，无则降级到 SwiftUI 占位）。
