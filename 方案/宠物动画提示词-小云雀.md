# Yoho 宠物动画 · 小云雀提示词工程

> 原则：先定锚（基准形象），再展开（四视图），后派生（场景动画）。
> 每一步用**相同的风格参考图**保证一致性。每切换品种时，复用上一个品种的风格锁定图。

---

## 阶段一：基准卡通形象设计

### 通用风格锁定提示词（4 个品种共用，每条提示词末尾必加）

```
【风格锁 - 所有品种强制附加】

可爱的卡通宠物角色，平面矢量插图风格。
干净统一的黑色描边（2-3px），柔和圆润的外形，温暖柔和的粉彩色调。
大眼睛带高光点，小巧简洁的嘴巴。
纯平涂、无阴影、无渐变、无立体感，纯赛璐珞 2D 游戏美术风格。
透明背景，全身入镜，站立或趴卧的中性姿态。
面向右前方（3/4 视角）。
分辨率：1024×1024。
```

### 1.1 银渐层

```
【01-银渐层-基准形象】

一只可爱的英短银渐层猫，卡通风格。
圆润紧凑的身体，短腿，大圆脸，脸颊饱满。
毛色：底层纯白（#F8F8F8），背部、头顶、尾巴有渐变灰黑色毛尖（#9B9B9B），
形成独特的"银色渐变"效果，胸口和肚皮为纯白。
眼睛：大圆形，明亮的翠绿色（#6DBE47），乖巧温顺的眼神，带两个白色高光点。
鼻子：小巧的砖红色倒三角（#D4786B）。
耳朵：中等大小，圆耳尖，耳内侧浅粉色，耳背毛色略深。
尾巴：粗且偏短，银灰色渐变，尾尖颜色最深。
表情：温顺、安静，眼神柔和，嘴角微微上扬，给人治愈感。
姿态：肚子贴地趴着，前爪蜷在胸口下，后爪向两侧摊开——经典的"板鸭趴"。

【风格锁 - 所有品种强制附加】
```

## 阶段二：四视图（每个品种生成一组）

### 通用四视图格式

```
【四视图模板 - 所有品种共用】

同一角色的四视图参考页。
相同的画风、相同的比例、相同的颜色。
排列为 2×2 网格：
  - 左上：正面图（正视图），面朝正前方
  - 右上：背面图（后视图），背对镜头
  - 左下：左侧面图（左视图），面朝左，侧身剪影
  - 右下：右前方四分之三视图（右3/4视图），面朝右前方
每张图：全身入镜，中性站立姿态，四肢自然垂放。
全部四个视图保持统一的描边粗细和色彩。
无背景。分辨率：2048×2048。
```

```
【05-银渐层-四视图】

【四视图模板 - 所有品种共用】
角色：参照【01-银渐层-基准形象】。
确保四个视图在以下方面完全一致：身体的圆润度、条纹图案的位置、
眼睛的大小和颜色、尾巴的粗细、耳朵的形状。
关键检查点：
- 正面：面部居中，额头条纹可见，肚皮奶油色斑块可见
- 背面：尾巴清晰可见，背部条纹图案完整，耳朵从背后视角
- 左侧面：身体侧面轮廓露出圆滚滚的肚皮弧线，尾巴自然下垂
- 右3/4视图：最常用的动画起始角度，面部约 70% 可见
```

## 阶段三：风格一致性验证

在生成动画之前，用以下提示词交叉验证 4 个品种风格统一：

```
【09-全家福-风格一致性检查】

Yoho 四只宠物并排站在同一张图里，从左到右依次为：
银渐层、布偶猫、泰迪犬、金毛犬。
全部面朝前方，中性站立姿态，同等缩放比例。
相同的平面矢量卡通风格，相同的描边粗细，相同的光照方式。
比例一致：猫的身高约为狗的 70%。
每只宠物通过其品种特征清晰可辨。
无背景。分辨率：3072×1024（横向全景）。
目的：验证四个角色是否属于同一个视觉宇宙。
```

## 阶段六：生成顺序（严格遵循）

```
【生成流水线】

第一步：依次生成【01】至【04】——4 张基准角色设计。
        → 审查：它们看起来像同一个"宇宙"里的角色吗？
        → 不像：用更强的风格锁定提示词重新生成。
        → 像：继续。

第二步：依次生成【05】至【08】——4 组四视图。
        → 审查：每个视图是否与步骤一的基准设计完全一致？
        → 重点检查：条纹/花纹/颜色在四个视图之间一致吗？
        → 不一致：单独重新生成不合格的视图。

第三步：生成【09】——风格一致性全家福检查。
        → 审查：比例、风格、颜色全部统一了吗？
        → 不统一：回到第一步，调整风格提示词后重新生成所有。

第四步：生成全部场景动画【4.1 至 4.10】。
        → 每条使用阶段二的右3/4视图作为起始帧参考图。
        → 每条末尾附上阶段五的【锚定短语】。
        → 每条注明阶段五色板中的精确十六进制颜色。

第五步：随机抽查 3 条动画输出，与阶段一基准形象对比。
        → 发现风格漂移：在锚定短语中加强措辞，重新生成受影响动画。
```
